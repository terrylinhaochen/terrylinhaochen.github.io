# Content Kit

This directory contains content guidelines, style guides, and reference materials to ensure consistency across all content creation.

## What Goes Here

### Style Guides
- Writing voice and tone guidelines
- Brand language preferences
- Content formatting standards
- Visual style requirements

### Reference Materials
- Key messaging documents
- Audience personas
- Content pillars and themes
- Editorial calendar templates

### Content Standards
- SEO guidelines
- Accessibility requirements
- Publication checklists
- Quality assurance criteria

## Usage

Claude should reference these materials when:
- Creating new content
- Reviewing drafts
- Ensuring brand consistency
- Making editorial decisions

## File Organization

Use descriptive names:
- `writing-style-guide.md`
- `brand-voice-guidelines.md`
- `content-review-checklist.md`
- `audience-personas.md`
