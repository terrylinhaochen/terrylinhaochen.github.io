This is an example of a post that has been archived. You can add the "archived" tag to any post you want to move to the archives section.

## Why Archive Notes?

Some notes might be:
- Historical references
- Outdated information you still want to preserve
- Reference material that's not currently active
- Personal notes you want to keep but not prominently display

## How to Use This System

1. Add the `archived` tag to any post you want to archive
2. The post will still appear in the main archives page
3. You can also create an archive-specific listing page if desired
